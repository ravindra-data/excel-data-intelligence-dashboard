# 📊 Data Intelligence Dashboard — Excel Final Project

An interactive and fully dynamic Excel Dashboard built for 2024-25 sales data — featuring 5 KPI cards, 4 charts, and multiple slicers for real-time filtering by Region, Category, Customer Segment, Year and Month.

- **Tool:** Microsoft Excel
- **Features:** Pivot Tables, Pivot Charts, Slicers, KPI Cards, Dynamic Dashboard, Conditional Formatting
- **Charts:** Line Chart, Bar Chart, Donut Chart, Horizontal Bar Chart
